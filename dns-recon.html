DNS Recon Ultra

Passo a passo para descobrir domínios e subdomínios a partir de IPs públicos

1. Ferramentas de Linha de Comando

1.1 dig (DNS Lookup)

dig -x 192.0.2.123 +short

Realiza uma consulta reversa DNS (PTR).

1.2 nslookup

nslookup 192.0.2.123

Outra forma de buscar o domínio reverso de um IP.

1.3 nmap

nmap -sV -Pn 192.0.2.123

Detecta serviços e, possivelmente, nomes de host via banners.

1.4 curl + APIs externas

curl https://api.hackertarget.com/reverseiplookup/?q=192.0.2.123

Consulta simples de domínios associados ao IP via API da HackerTarget.

curl -H "apikey: SUA_API_KEY" https://api.securitytrails.com/v1/search/list -d '{"filter":{"ipv4":"192.0.2.123"}}'

Consulta mais detalhada via SecurityTrails.

2. Ferramentas e Serviços Online

  • VirusTotal IP Lookup
    Consulta domínios, certificados SSL, roteamento e relações históricas com o IP.
  • SecurityTrails IP Search
    Histórico de domínios relacionados ao IP, com informações WHOIS e DNS.
  • Shodan Host Info
    Exibe serviços e domínios vinculados ao IP com base em fingerprint de rede.
  • ViewDNS Reverse IP Lookup
    Consulta reversa gratuita baseada em registros DNS públicos.
  • DNSDumpster
    Plataforma visual que retorna domínios, subdomínios e servidores expostos.

3. Dica Extra

Para grandes volumes de IPs, utilize scripts em bash com curl e jq, ou ferramentas como DNS Recon Ultra, que já automatizam todo o processo.
